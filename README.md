# Nikil Abbishak's Portfolio

A modern, responsive, and beautifully designed personal portfolio website built with React and Vite.

## 🚀 Live Demo

You can view the live version of this portfolio here:
[**https://Nikil-Abbishak.github.io/Portfolioo-SAMPLE/**](https://Nikil-Abbishak.github.io/Portfolioo-SAMPLE/)

## 🛠️ Built With
- **React** - Component-based UI development
- **Vite** - Lightning-fast frontend tooling
- **CSS3** - Custom styling and animations

## ⚙️ Running Locally

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the local development server

## 🌐 Deployment
This project is configured to deploy directly to GitHub Pages using the `gh-pages` package. 
To deploy a new version, simply run:
```bash
npm run deploy
```
